Displaying Data from Multiple Tables Using Joins - Hands-on Assignments

Assignment 1: Write a query for the HR department to produce the addresses of all the departments. Use the EMP and DEPT tables. Show the EMPNO, ENAME,SAL, DNAME and LOC in the output. Use a NATURAL JOIN to produce the results.

SELECT Empno, Ename, Sal, Dname, Loc 
FROM Emp NATURAL JOIN Dept;


Assignment 2: The HR department needs a report of all employees. Write a query to display the JOB,MGR,SAL,COMM,DNAME of employees whose JOB is SALESMAN.

Using an Equi Join (INNER JOIN) on Deptno
SELECT e.Job, e.MGR, e.Sal, e.Comm, d.Dname 
FROM Emp e JOIN Dept d ON e.Deptno = d.Deptno 
WHERE e.Job = 'SALESMAN';


Assignment 3: The HR department needs a report of employees in LOC DALLAS. Display the ENAME, job, DEPTNO, and DNAME for all employees who work in DALLAS.

SELECT e.Ename, e.Job, e.Deptno, d.Dname 
FROM Emp e JOIN Dept d ON e.Deptno = d.Deptno 
WHERE d.Loc = 'DALLAS';


Assignment 4: Create a report to display employees' ename and employee number along with their manager's name and manager number. Label the columns Employee, Emp#, Manager, and Mgr#, respectively.

Using a Self Join on the Emp table (joining it to itself)
SELECT e.Ename AS "Employee", 
       e.Empno AS "Emp#", 
       m.Ename AS "Manager", 
       m.Empno AS "Mgr#" 
FROM Emp e JOIN Emp m ON e.MGR = m.Empno;


Assignment 5: Modify the previous Query to display all employees including King, who has no manager. Order the results by the employee number.

Using a LEFT OUTER JOIN so we keep employees even if they don't have a matching manager record
SELECT e.Ename AS "Employee", 
       e.Empno AS "Emp#", 
       m.Ename AS "Manager", 
       m.Empno AS "Mgr#" 
FROM Emp e LEFT OUTER JOIN Emp m ON e.MGR = m.Empno 
ORDER BY e.Empno;


Assignment 6: The HR department needs a report on job grades and salaries. To familiarize yourself with the SALGRADE table, first show the structure of the SALGRADE table. Then create a query that displays the name, job, department name, salary, and grade for all employees.

Showing structure first
DESC SALGRADE;

Now doing a Non-Equi Join using BETWEEN for the salary grades, and an Equi Join for the department
SELECT e.Ename, e.Job, d.Dname, e.Sal, s.Grade 
FROM Emp e 
JOIN Dept d ON e.Deptno = d.Deptno 
JOIN Salgrade s ON e.Sal BETWEEN s.Losal AND s.Hisal;


Assignment 7: Display the ENAME,DNAME of all the employees. Also display those department name which do not have any employees working.

Using a RIGHT OUTER JOIN to ensure all departments are listed even if no employees match
SELECT e.Ename, d.Dname 
FROM Emp e RIGHT OUTER JOIN Dept d ON e.Deptno = d.Deptno;


 Assignment 8: The HR department needs to find the names and hire dates for all employees who were hired before their managers, along with their managers' names and hire dates. (Self Join with Outer Join)

 Using a LEFT OUTER JOIN for the self join as requested by the topic hint, and filtering by hire date in the WHERE clause.
SELECT e.Ename AS "Employee Name", 
       e.Hiredate AS "Emp Hire Date", 
       m.Ename AS "Manager Name", 
       m.Hiredate AS "Mgr Hire Date" 
FROM Emp e 
LEFT OUTER JOIN Emp m ON e.MGR = m.Empno 
WHERE e.Hiredate < m.Hiredate;


Assignment 9: Display the EMPNO,ENAME,DNAME,LOC of those employees who are working as CLERK. Use the USING clause.

USING clause simplifies the join condition when column names match exactly.
SELECT Empno, Ename, Dname, Loc 
FROM Emp 
JOIN Dept USING (Deptno) 
WHERE Job = 'CLERK';


Assignment 10: Display the ENAME,SAL,MGR,DNAME of employees whose salary is more than 2000. Use the ON clause.

SELECT e.Ename, e.Sal, e.MGR, d.Dname 
FROM Emp e 
JOIN Dept d ON e.Deptno = d.Deptno 
WHERE e.Sal > 2000;


Assignment 11: Display the EMPNO,ENAME,JOB,DEPTNO,DNAME,LOC of employees. Use LEFT OUTER JOIN.

SELECT e.Empno, e.Ename, e.Job, e.Deptno, d.Dname, d.Loc 
FROM Emp e 
LEFT OUTER JOIN Dept d ON e.Deptno = d.Deptno;


Assignment 12: Display the ENAME,DNAME of employees. Use RIGHT OUTER JOIN.

This will ensure all departments are shown, even if they have no employees.
SELECT e.Ename, d.Dname 
FROM Emp e 
RIGHT OUTER JOIN Dept d ON e.Deptno = d.Deptno;


Assignment 13: Display the EMPNO,DNAME,LOC of employees. Use FULL OUTER JOIN.

This grabs everything from both tables, matching where possible and returning nulls where not.
SELECT e.Empno, d.Dname, d.Loc 
FROM Emp e 
FULL OUTER JOIN Dept d ON e.Deptno = d.Deptno;
