Reporting Aggregated Data Using the Group Functions - Hands-on Assignments


Assignment 1: Find the highest, lowest, sum, and average salary of all employees. Label the columns

The prompt cuts off here, so I'll just provide the basic aggregate functions.
SELECT MAX(Sal), 
       MIN(Sal), 
       SUM(Sal), 
       AVG(Sal) 
FROM Emp;


Assignment 2: Maximum, Minimum, Sum, and Average, respectively. Round your results to the nearest whole number.

 Applying the labels and rounding to the previous query's requirements.
SELECT ROUND(MAX(Sal)) AS "Maximum", 
       ROUND(MIN(Sal)) AS "Minimum", 
       ROUND(SUM(Sal)) AS "Sum", 
       ROUND(AVG(Sal)) AS "Average" 
FROM Emp;


Assignment 3: Modify the above query to display the minimum, maximum, sum, and average salary for each job type.

Adding the Job column and the GROUP BY clause to aggregate per job type
SELECT Job, 
       ROUND(MAX(Sal)) AS "Maximum", 
       ROUND(MIN(Sal)) AS "Minimum", 
       ROUND(SUM(Sal)) AS "Sum", 
       ROUND(AVG(Sal)) AS "Average" 
FROM Emp 
GROUP BY Job;


Assignment 4: Write a query to display the number of people with the same job

COUNT(*) will count all rows per job group
SELECT Job, COUNT(*) AS "Number of People" 
FROM Emp 
GROUP BY Job;


 Assignment 5: Determine the number of managers without listing them. Label the column Number of Managers

 Using COUNT(DISTINCT MGR) so we only count each manager ID once
SELECT COUNT(DISTINCT MGR) AS "Number of Managers" 
FROM Emp;


Assignment 6: Find the difference between the highest and lowest salaries. Label the column DIFFERENCE.

SELECT MAX(Sal) - MIN(Sal) AS DIFFERENCE 
FROM Emp;


Assignment 7: Create a report to display the manager number and the salary of the lowest-paid employee for that manager. Exclude anyone whose manager is not known. Exclude any groups where the minimum salary is $2000 or less. Sort the output in descending order of salary.

WHERE filters out null managers before grouping, HAVING filters the groups based on the minimum salary
SELECT MGR, MIN(Sal) 
FROM Emp 
WHERE MGR IS NOT NULL 
GROUP BY MGR 
HAVING MIN(Sal) > 2000 
ORDER BY MIN(Sal) DESC;


Assignment 8: Create a query to display the total number of employees and, of that total, the number of employees hired in 1980, 1981, and 1982. Create appropriate column headings.

Using SUM with DECODE to act like a pivot table for the different years
SELECT COUNT(*) AS "Total",
       SUM(DECODE(TO_CHAR(Hiredate, 'YYYY'), '1980', 1, 0)) AS "1980",
       SUM(DECODE(TO_CHAR(Hiredate, 'YYYY'), '1981', 1, 0)) AS "1981",
       SUM(DECODE(TO_CHAR(Hiredate, 'YYYY'), '1982', 1, 0)) AS "1982"
FROM Emp;
