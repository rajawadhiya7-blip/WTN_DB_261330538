Topic: Retrieving Data Using the SQL SELECT Statement - Hands-on Assignments

Assignment 1: Determine the Structure of DEPT Table and its Contents.

Solution-

using DESC command to see the structure of the DEPT table
DESC DEPT;

using select * to see all the data inside it
SELECT * FROM DEPT;


Assignment 2: Determine the Structure of EMP Table and its Contents.

Solution-
looking at the EMP table structure first
DESC EMP;

now viewing what's inside the EMP table
SELECT * FROM EMP;


Assignment 3: Display the Ename and Deptno from Emp table whose Empno is 7788.

Solution-
selecting just the employee name and department number for the specific employee ID
SELECT Ename, Deptno 
FROM Emp 
WHERE Empno = 7788;
